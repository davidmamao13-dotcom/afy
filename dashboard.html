O que é o AFY?

O AFY AutoBot é um robô que publica vídeos automaticamente em múltiplas redes sociais — Instagram, TikTok, Facebook, YouTube Shorts e Pinterest.

Sem API Keys: Usa navegador real (Chrome) — você faz login 1 vez e o robô salva a sessão
IA Generativa: Legendas e hashtags automáticas (Gemini grátis ou ChatGPT)
Cross-Posting: Um vídeo publicado nas contas de vários afiliados
Agendamento: Posta automaticamente nos horários que você definir
Respostas Inteligentes: IA responde comentários automaticamente
afy@bot:~$
# 🚀 AFY AutoBot
# Automação sem API Keys!

from afy import AutoBot

bot = AutoBot()
bot.login("Instagram") # Login 1x
bot.post("video.mp4") # Posta automático
bot.schedule("08:00, 12:00, 18:00")
bot.auto_reply() # IA responde
