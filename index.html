O que é o AFY?

O Robô que
Posta por Você

O AFY é um sistema inteligente de automação para afiliados digitais. Ele publica vídeos automaticamente em múltiplas redes sociais, gera legendas com IA e responde comentários — sem precisar de API keys.

Login Assistido: Você faz login 1 vez no Chrome, o Bot salva a sessão.
Cross-Posting: Um vídeo publicado na conta de vários afiliados.
IA Integrada: Legendas, hashtags e respostas geradas automaticamente.
Agendador: Defina os horários e o robô posta automaticamente.
# AFY - Automação Inteligente # Sem API Keys! 😎

from afy import AutoBot

bot = AutoBot()
bot.login("Instagram") # Login 1x
bot.post("video.mp4") # Posta automático
bot.schedule("08:00, 12:00, 18:00")
bot.auto_reply() # IA responde

# 📤 Publicando em 5 redes... # ✅ 127 vídeos publicados # 💰 R$ 3.2K em comissões
